# API: adminNavcatModify


## ACCESS REQUIREMENTS: ##
[[NAVCAT|CONCEPT_navcat]] - U


changes the pretty name of a navigation category or list

[Concept - navcat](concept_navcat)

## INPUT PARAMETERS: ##
  * path: .safe.name or path:$listname
  * pretty: new name for category
