Create a Product List

USAGE: Enter the List name you wish to create. List names may be a combination of both upper and lower case characters and numbers, punctuation, or other characters. When you are done press the SAVE button. Simply put, lists are just another way of grouping products together without needing to create a category.

Lists can be referenced on multiple pages. For example you might use a list to create a "Recommended Products", or "Best Sellers" list that can be displayed by both products, as well as in the categories those products are displayed in. (Hint: it is our intention that Zoovy will eventually create and automatically update a variety of lists for you.)

SPECIAL CONSIDERATIONS:

TIP HIDDEN LISTS: If you don't want the name of the list to be displayed, or the products in the list to be indexed then in search then you should pre-pend an exclamation mark (!) as the very first character the list name.

META PARAMETERS: Using Meta Parameters it is possible to create lists which map to a search results, or even another store, or ebay listings. Example: CATALOG=DEFAULT&QUERY=string
