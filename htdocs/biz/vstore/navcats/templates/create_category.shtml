SPECIAL CONSIDERATIONS: Certain themes may not properly render all punctuation characters. Make sure to review your website after making changes.

TIP HIDDEN CATEGORIES: Categories may be hidden by placing a exclamation mark (!) as the first character of any name. Hidden categories will NOT appear on the website, but will appear in searches. For example !Specials will not appear on the website, but could be linked to directly in an email newsletter for certain customers.

TIP SORTING CATEGORIES: Categories are sorted in alpha-numeric order based on the name they are created with. If you would like categories to appear in a certain order, then you need to plan ahead and initially create the categories as numbers (e.g. 1, 2, 3, 4) then rename them to the order you would like. In the future if you need to add a category between 1 and 2, then simply create a 1A and it will automatically go between categories 1 and 2.
