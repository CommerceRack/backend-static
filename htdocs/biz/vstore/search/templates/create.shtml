Catalog Code: [?]
Catalog Code: is an alphanumeric code that will be used by you to uniquely identify this catalog. Later on when you are asked to select a catalog for a particular wrapper or page layout you will be able to choose from a list of codes. Usually the first catalog in a store is named "DEFAULT" but really any text will do.
