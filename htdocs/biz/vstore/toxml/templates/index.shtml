Templates Manager

Overview
All pages hosted by Zoovy are generated by our CMS (Content Management System). The CMS uses templates to determine what information is displayed on the page, and how it is structured.

Templates make it easy to keep HTML formatting consistent, which ultimately leads to increased sales.

Output Formats
There are three different Output Formats available for each template, all formats contain identical information formatted slighly differently.
Properly formatted XML, useful for loading into another software application or XML editor.
HINT: More advanced documents (which contain other embedded documents) may be easier to edit in this format.


Template Purposes
Each template must have a purpose. Purpose ensures that an end-user who is operating Zoovy can only choose an appropriate template, for an appropriate situation.

Wrapper
Used to control the overall site look and appearance.

Layout
Fits within the wrapper, this is the template which controls the page specific content.

Wizard
Used for creating the HTML in Auction or Marketplace listings.

Definition
Used to customize the data input fields for a particular marketplace.

Email Files
Base email templates that can be edited in the page wizard.

Order Files
Base order templates that can be edited in the page wizard.

Include File
Utility files which can be loaded from any other file type.


Hint: now that you understand how the template purposes, and output formats you can select from the Tabs along the top of the screen to go to the appropriate purpose.
